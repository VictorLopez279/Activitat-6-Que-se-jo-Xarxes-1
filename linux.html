Xarxa en Linux (Debian i Ubuntu 24.04)

IniciLinux

Debian (ifupdown o /etc/network/interfaces)

# Exemple /etc/network/interfaces (ifupdown)
auto eth0
iface eth0 inet static
    address 192.168.10.20
    netmask 255.255.255.0
    gateway 192.168.10.1
    dns-nameservers 8.8.8.8 8.8.4.4

Ubuntu 24.04 (netplan)

# /etc/netplan/01-network.yaml
network:
  version: 2
  ethernets:
    enp0s3:
      dhcp4: no
      addresses: [192.168.10.30/24]
      gateway4: 192.168.10.1
      nameservers:
        addresses: [8.8.8.8,8.8.4.4]

# Aplicar
sudo netplan apply

Comprovar connectivitat: ip a, ip route, ping 8.8.8.8
